# 🔍 Network Port Scanner

> 🚀 Beginner-friendly project for learning networking and cybersecurity basics

## 📌 Overview
This project is a simple Python GUI application that scans open TCP ports on a target system.  
This project was developed as part of an internship to understand networking concepts practically.

## 🚀 Features
- Scan open ports on a target IP or hostname  
- Simple GUI using Tkinter  
- Faster scanning using multithreading  
- Displays open ports with service names  
- Save scan results to a file  
- Start and stop scan functionality  

## 🛠️ Technologies Used
- Python  
- Tkinter  
- Socket Programming  
- Threading  

## ⚙️ How It Works
- User enters target IP and port range  
- Program tries to connect to each port  
- If connection is successful, port is open  
- Results are displayed in the GUI  
- Threading is used to speed up scanning  

## ▶️ How to Run
1. Download the project  
2. Open terminal in the folder  
3. Run:

python main.py
