# 🔍 Network Port Scanner

## 📌 Overview
This project is a simple Python GUI application that scans open TCP ports on a target system. It helps users understand basic networking and port scanning in an easy way.

## 🚀 Features
- Scan open ports on a target IP or hostname  
- Simple graphical interface using Tkinter  
- Faster scanning using multithreading  
- Displays open ports with service names  
- Save scan results to a text file  
- Start and stop scan functionality  

## 🛠️ Technologies Used
- Python  
- Tkinter  
- Socket Programming  
- Threading  

## ▶️ How to Run
1. Download the project files  
2. Open terminal in the project folder  
3. Run the command: python main.py
